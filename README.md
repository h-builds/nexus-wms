# NexusWMS

NexusWMS is an agentic warehouse orchestration portfolio project designed for 2026 logistics workflows.

## Apps

- `apps/api`: Laravel 13 backend
- `apps/vapor-monitor`: real-time monitoring dashboard
- `apps/field-agent-mobile`: mobile operational capture
- `apps/orchestrator-twin`: tactical simulation layer

## Principles

- Modular monolith backend
- Domain-driven folder boundaries
- Offline-first field operations
- Event-driven state propagation
- AI governance by design
